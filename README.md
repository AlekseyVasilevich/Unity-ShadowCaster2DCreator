Скрипт для автоматического создания теней на Tilemap 2D Unity

Для коректной работы на объекте должны быть следующие компоненты:
1) Tilemap Collider 2D
2) Rigidbody 2D (Body Type = Static)
3) Composite Collider 2D

Для создания теней:
1) Переместите скрипт на объект
2) Заберите галочку с Self Shadows
3) Нажмите: "Create"

Для добавления теней на новые объекты необходимо удалить тени (Remove Shadows) и заново создать тени (Create)

Script for automatically generating shadows on a 2D Tilemap in Unity

For the script to work properly, the object must have the following components:
1) Tilemap Collider 2D
2) Rigidbody 2D (Body Type = Static)
3) Composite Collider 2D

To create shadows:
1) Attach the script to the object
2) Uncheck the “Self Shadows” box
3) Click “Create”

To add shadows to new objects, you must first remove the shadows (Remove Shadows) and then recreate them (Create)
